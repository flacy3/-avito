Минимальные тесты Avito

Простой проект для проверки работоспособности Avito
основные пользовательские сценарии Avito

1. Переход на страницу - Переходит сразу на страницу конкретного товара
2. Поиск кнопки избранного - Ждет появления кнопки "В избранное"
3. Запись начального состояния - Сохраняет CSS-класс кнопки (например: "favorite-btn")
4. Нажатие на кнопку - Добавляет товар в избранное
5. Проверка изменения - Ждет изменения класса кнопки
6. Финальная проверка - Проверяет новый класс кнопки
